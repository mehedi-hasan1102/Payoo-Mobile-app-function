# Payoo - Mobile Banking App Frontend

## Overview
**Payoo** is a **mobile banking frontend application** built using **HTML, JavaScript, and Tailwind CSS**. This project demonstrates a functional and responsive mobile app interface for banking operations, developed as a frontend-only application.  

The focus is on responsive design, interactive UI elements, and a smooth user experience for mobile devices.

## Technologies Used
- HTML5
- JavaScript (Vanilla JS)
- Tailwind CSS
- Mobile-first responsive design

## Features
- Responsive mobile app layout
- Interactive UI components using JavaScript
- Clean and modern design with Tailwind CSS
- Simulated banking app interface (frontend only)
